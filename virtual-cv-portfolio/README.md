# Thabo Nyathela | Virtual CV Portfolio

A professional one-page virtual CV and portfolio website for Thabo Nyathela, built with HTML5, CSS3, and vanilla JavaScript for direct deployment to GitHub Pages.

## Live Site

GitHub Pages placeholder:
https://github.com/thabonyathela/virtual-cv-portfolio

## Features

- Responsive single-page portfolio with smooth-scroll navigation
- Sticky navigation bar with active section highlighting
- Dark/light mode toggle using CSS variables and JavaScript
- Scroll reveal animations and hover effects for cards
- Front-end-only contact form with user feedback
- Suitable for zero-configuration deployment on GitHub Pages

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts CDN

## Setup and Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/thabonyathela/virtual-cv-portfolio.git
   ```
2. Open the project folder:
   ```bash
   cd virtual-cv-portfolio
   ```
3. Start a simple local server:
   ```bash
   python -m http.server 8000
   ```
4. Open the site in your browser at:
   ```text
   http://localhost:8000
   ```
